MySchedule
==========

A Schedule application with user autentication with just hash saved and schedule notification.

Android Features

* Fragment
* Transaction Animation
* Shared Preferences
* SQLite Database
* Content Provider
* Adapter
* Loader
* Intent Action PickContact
* Intent Filter with MimeType
* Service
* Notification
* Broadcast Receiver

Dependences

* RecyclerView by Google
* CardView by Google
* Gson by Google
* Picasso by SquareUp
* FloatingActionButton by Melnykov
